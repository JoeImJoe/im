# 消息流组件 

用于显示消息，目前包含以下组件：

- MessageText
- MessageMedia（ 图片 ）

## 基础API（ MessageBase ）

| Name     | Type            | Default | Description  |
| :------- | :-------------- | :------ | :----------- |
| reverse  | bool            | `false` | 是否反向显示 |
| avatar   | string          |         | 头像地址     |
| name     | string          |         | 用户名称     |
| children | React.ReactNode | null    | 聊天内容     |
